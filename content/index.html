This small game is intended to demonstrate capabilities of Kotlin/Native software stack.

Rules of the game

  • Download and install mobile application for or .
  • System will assign you random team, marked with the color.
  • Spin logo ot shake your phone.
  • Each shake or full rotation will increment your team's score.
  • Team with biggest score at the end of conference wins.
  • Most active contributors of winning team will get special prizes from us.

Technical details

  • Whole stack is implemented in Kotlin/Native.
  • Server side is implemented on top of microHTTPD HTTP server library, SQLite as database storing score, and runs on Linux server.
  • Client side for is implemented in pure Kotlin/Native as an Native Activity, using GLES interop for rendering.
  • Client side for is implemented in pure Kotlin/Native using full AppKit integration.
  • Client/Server interaction works via JSON requests over HTTP, using Jansson JSON library and libcurl.
  • Server interaction is asynchronous from the UI thread, using workers.
  • HTTP server works in multithreaded mode, state shared between threads is implemented via SQLite DB access.
